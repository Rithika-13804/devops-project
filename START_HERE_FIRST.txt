## 🎯 UNDERSTANDING THE FLOW

### Adding a Book (Example Flow)
```
User clicks "Add Book" → Dashboard screen closes
                    ↓
        AddBook.fxml is displayed
                    ↓
User fills form and clicks "Add Book"
                    ↓
    AddBookController.onAddBook() is called
                    ↓
Input validation happens (checks required fields)
                    ↓
bookDAO.addBook() is called
                    ↓
Book object is created and added to ArrayList
                    ↓
Success message shown "✅ Book added successfully!"
                    ↓
Form clears automatically
                    ↓
User can add more books or go back to dashboard
```

### Issuing a Book (Example Flow)
```
User clicks "Issue Book" → IssueReturn.fxml is displayed
                    ↓
User selects book from dropdown (only available books)
                    ↓
User selects student from dropdown
                    ↓
User clicks "Issue Book"
                    ↓
IssueReturnController.onIssueBook() is called
                    ↓
Validation: Check if book is available
                    ↓
Two operations happen:
  1. issueRecordDAO.issueBook() - creates issue record
  2. bookDAO.issueBook() - marks book as issued
                    ↓
Book status changes from "Available" to "Issued"
                    ↓
Book removed from available books dropdown
                    ↓
Success message shown
```
